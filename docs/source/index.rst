=========================================================
Example of Sphinx documentation for a Common Lisp library
=========================================================

This is a small library includes a few functions with docstrings and a handwritten documentation.
The purpose is to demonstrate core features of the
`sphinxcontrib-cldomain <https://github.com/40ants/sphinxcontrib-cldomain>`_ library.

The repository can be used as a template for new libraries if you've choosen ``cldomain``
for documenting your library.

Let's review features, provided by ``cldomain``.
